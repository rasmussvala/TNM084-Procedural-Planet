# TNM084-Procedural-Planet

## Overview

This THREE.js project, part of the  [TNM084 Procedural Methods for Images](https://studieinfo.liu.se/kurs/TNM084/ht-2024) course at Link√∂ping University, generates procedurally created planets. It utilizes various types of noises to produce unique landscapes and scenery.

## Usage

### Prerequisites

- Node.js
- npm (Node Package Manager)

### Setup

1. Clone the repository.
2. Navigate to the project directory by running the command: ```cd TNM084-Procedural-Planet```
3. Install dependencies by running the command: ```npm install```
4. Start the project by running the command: ```npx vite```
   
You should now be able to access the project in a web browser at 'http://localhost:XXXX'.
